CureCast - Disease Prediction and Remedy Recommendation System
Overview
CureCast is an intelligent web application that predicts diseases based on user-provided symptoms and recommends appropriate remedies using machine learning algorithms.

Features
Accurate disease prediction
Personalized remedy recommendations
User-friendly interface
Data visualization for predictions
Installation
Clone the repository:
git clone https://github.com/Ekanshiganglas/Disease_predictions_and_Remedy_recommendation_project/
Disease-Prediction-and-Remedy-Recommendation
Install dependencies:
pip install -r requirements.txt
Run the application:
python app.py
Usage
Enter symptoms in the input form
View predicted disease and recommended remedies
Tech Stack
Python
Flask
scikit-learn
Pandas
Model Details
Models trained on public health datasets using supervised learning algorithms.
