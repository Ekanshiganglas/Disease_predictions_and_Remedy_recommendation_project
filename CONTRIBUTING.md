# Contributing to CureCast 🏥

Thank you for your interest in contributing to CureCast!
All contributions are welcome and appreciated.

---

## 🐛 Reporting a Bug

If you found a bug:

1. Go to the **Issues** tab on GitHub
2. Click **New Issue**
3. Write:
   - What you were doing
   - What you expected to happen
   - What actually happened
   - Your operating system and Python version

---

## 💡 Suggesting a Feature

Have an idea to improve CureCast?

1. Go to the **Issues** tab on GitHub
2. Click **New Issue**
3. Start the title with `[Feature Request]`
4. Describe your idea clearly

---

## 🛠️ Contributing Code

### Step 1 — Fork the repository
Click the **Fork** button on the top right of the repo page.
This creates your own copy of the project.

### Step 2 — Clone your fork
```bash
git clone https://github.com/YOUR_USERNAME/CureCast.git
cd CureCast
```

### Step 3 — Create a new branch
```bash
git checkout -b feature/your-feature-name
```
Never work directly on the `main` branch.

### Step 4 — Make your changes
Edit the code. Test it. Make sure everything works.

### Step 5 — Commit your changes
```bash
git add .
git commit -m "feat: describe what you changed"
```

### Step 6 — Push to your fork
```bash
git push origin feature/your-feature-name
```

### Step 7 — Open a Pull Request
- Go to the original CureCast repo on GitHub
- Click **Pull Requests** tab
- Click **New Pull Request**
- Describe what you changed and why

---

## ✅ Code Style Guidelines

- Follow **PEP 8** for Python code
- Use meaningful variable names
- Add comments for non-obvious logic
- Keep functions small and single purpose
- Test your changes before submitting
